## 📁 Projects Included

### 🔹 1. COVID-19 Data Analysis

**Objective:**
Analyze COVID-19 data to understand trends in cases, deaths, recoveries, and testing.

**Key Tasks:**

* Data cleaning (handling missing values)
* Trend analysis of cases and deaths
* Visualization of COVID spread

**Insights:**

* Identified growth patterns of cases
* Compared recovery vs death rates
* Analyzed testing impact

---

### 🔹 2. House Prices Analysis

**Key Tasks:**

**Insights:**

* Relationship between area, location, and price

### 🔹 3. Weather Data Analysis

## 📌 How to Run the Projects

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Data_Analysis_Portfolio.git
   ```

2. Navigate to the folder:

   ```bash
   cd Data_Analysis_Portfolio
   ```

3. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
